# SLD_ALGORITHM_LAYOUT_SPEC

## Reguły geometryczne
- GPZ: kotwica górna.
- Trunk: oś pionowa, porządek zgodny z kolejnością domenową.
- Branch: odejście poziome z trunk + prowadzenie pionowe.
- Station block: zwarty układ SN/nN (szyna SN, pole trafo, transformator, szyna nN).
- Ring/NOP: kanał wtórny, bez zmiany geometrii trunk.

## Determinizm
- Identyczna topologia wejściowa musi dawać identyczne koordynaty bazowe.
- Brak zależności od kolejności przypadkowej tablic wejściowych.
