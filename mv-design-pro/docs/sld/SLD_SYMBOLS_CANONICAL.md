# SLD_SYMBOLS_CANONICAL

## Minimalny zestaw symboli kanonicznych
- GPZ/Source
- Bus SN / Bus nN
- LineBranch / CableBranch
- TransformerBranch
- Switch / Breaker / Disconnector / Earthing
- CT / VT / Relay
- Ring link / NOP marker
- PV/BESS przez transformator

## Zasady
- Renderer nie może podmieniać symbolu na fallback zmieniający semantykę topologiczną.
- Label anchors pochodzą z geometrii bazowej.
